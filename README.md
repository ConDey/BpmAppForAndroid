1. 应用插件动态化
2. 更新系统升级模块，使之更人性化

2. 首页的APP列表
3. 通讯录
4. 通知公告
5. 修改密码
6. 通用的jswebview
7. 文件选择器

9. 系统升级

1. 关闭窗口
2. 支持维护Toolbar
3. 绑定退出键

5. 拍照/选取照片
6. 视频/选取视频
7. 选择本地文件，本地文件管理器
8. 上传/下载文件,并显示进度
